# sounds フォルダ

タイマーが 0:00 になったときに再生される音声ファイル（MP3）を、このフォルダに置いてください。

## 使い方

1. 再生したい `.mp3` ファイルをこのフォルダに追加する。
2. `manifest.json` の `"sounds"` 配列に、追加したファイル名を列挙する。

### manifest.json の例

```json
{
  "sounds": [
    "alarm1.mp3",
    "alarm2.mp3",
    "fanfare.mp3"
  ]
}
```

タイマーが終了すると、`manifest.json` に列挙されたファイルの中からランダムに 1 つが選ばれ、**Stop が押されるまでループ再生**されます。  
複数タイマーが同時に終了した場合、それぞれが独立して音を再生します（重なって鳴ります）。

## 注意事項

- ファイルはブラウザが再生できる形式（主に MP3）を推奨します。
- GitHub Pages に公開する場合は、音声ファイルも一緒にコミット＆プッシュしてください。
- `manifest.json` を更新しないと、新しく追加したファイルは再生されません。
- ブラウザのオートプレイ制限により、ページを開いた直後にユーザー操作なしでは音が鳴らない場合があります。タイマーの Start ボタンを一度押していれば問題なく動作します。
